# 项目列表

## 核心目标
快速浏览、搜索、过滤、跳转所有项目；支持新建与删除。

## 线框

```
┌────────────────────────────────────────────────────────────────────────┐
│  项目                                                  [ + 新建项目 ]  │
├────────────────────────────────────────────────────────────────────────┤
│  [搜索 🔍] [Target ▾] [类型 ▾] [最近活跃 ▾]                            │
├────────────────────────────────────────────────────────────────────────┤
│  名称           │ 类型      │ Target   │ MC      │ 最近构建    │ ⋯   │
│  MyAwesomePlugin│ plugin    │ paper    │ 1.20.4  │ ● success   │ ⋯   │
│  ProxyChat      │ proxy     │ velocity │ 1.21.1  │ ● running   │ ⋯   │
│  ServerBalancer │ plugin    │ folia    │ 1.21.1  │ —           │ ⋯   │
│  EconomyMod     │ mod       │ fabric   │ 1.20.6  │ ✗ failed    │ ⋯   │
└────────────────────────────────────────────────────────────────────────┘
```

## 行操作菜单 (⋯)

- 打开详情
- 构建
- 复制项目
- 导出 spec
- 删除（需二次确认）

## 数据

- `GET /api/projects?q=&targetId=&type=&sort=`
- 服务端过滤；不在前端做大列表过滤
- 分页：默认每页 20

## 状态

- **空**：大插画 + "你还没有项目，点击右上角新建" CTA
- **加载**：表格骨架
- **错误**：横幅 `[ERROR_CODE] messageZh + 重试按钮`
- **删除**：模态二次确认 + 输入项目 slug 解锁删除按钮

## 关键交互

| 触发 | 行为 |
|------|------|
| 点击行 | 跳转 `/projects/:id` |
| 点击列头 | 切换排序 |
| 删除 | DELETE /api/projects/:id，乐观更新 |
| 新建 | 跳转 `/projects/new` |

## 可访问性

- 表格使用 `role="table"` + `aria-sort`
- 删除模态焦点陷阱
