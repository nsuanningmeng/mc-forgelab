## 字段持久化

| 字段 | 存储 | 范围 |
|------|------|------|
| workspace/cache/db 路径 | 仅显示（来自环境变量） | 只读 |
| HTTP 代理 / 镜像 | settings 表 | 用户级 |
| 构建并发数 | settings 表（env 优先） | 服务级 |
| 产物保留 | settings 表（env 优先） | 服务级 |
| 认证启用 | env 重启生效（**保存按钮变 "需要重启"**） | 服务级 |
| 主题 / 语言 | localStorage + settings | 用户级 |

## 安全

- **绝不**在 UI 显示已设置的密码（仅显示 "已设置"）
- 修改密码必须经 CLI：`mcforgelab admin set-password`（阶段 6+）
- 镜像 URL 必须 HTTPS（HTTP 给警告但允许）

## 兼容性提示

- 若 `MC_FORGELAB_BASE_URL` 是 https 且 `MC_FORGELAB_AUTH_ENABLED=false` → 顶部红色 banner "公网部署建议开启登录"
