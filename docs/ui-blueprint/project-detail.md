## 操作菜单 (⋯)

- 删除项目
- 复制为新项目
- 导出 spec JSON
- 在文件管理器中显示（桌面端）

## 数据

- `GET /api/projects/:id`
- `GET /api/projects/:id/tree?depth=3`
- `GET /api/projects/:id/builds?limit=5`

## 状态

- **空文件树**：项目刚创建未渲染（点击 "[ 重新生成 ]"）
- **构建按钮**：若有 running 构建则禁用，文案改为 "构建中..."
- **删除**：模态二次确认 + 输入项目 slug
