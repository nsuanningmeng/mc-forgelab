## 核心目标
让用户**不需要自己装** Java / Gradle / Maven 即可开始构建；可视化检测、安装、校验。

## 安装流程

点击 [安装] → 模态：

```
安装 Java 17 (Adoptium)
─────────────────────────
平台：linux x64
源：https://api.adoptium.net/...（HTTPS）
大小：180 MB
SHA256：a1b2c3...（下载后自动校验）

进度：████░░░░░░░░ 30%   54 / 180 MB
状态：下载中

[ 取消 ]
```

完成 → 自动 `verify` → manifest 落盘 → 状态切 `validated`。

## 验证

- 比对 manifest.json 的 sha256
- 执行 `<exe> -version` 验证启动
- 失败 → 状态 `invalid` + 操作 [ 重新安装 ]

## 离线/受限网络

- 支持手动指定本地 zip 路径（拖拽到工具链页）
- sha256 校验后导入

## 数据

- `GET /api/toolchains` → ToolchainManifestEntry[]
- `POST /api/toolchains/install` → 异步 jobId
- `POST /api/toolchains/verify`
- `GET /api/toolchains/doctor`

## 安全

- 安装目录限制在 `MC_FORGELAB_TOOLCHAINS` 内
- 解压使用 stream，**校验路径不逃逸**（防 zip slip）
- 不写入系统 PATH
