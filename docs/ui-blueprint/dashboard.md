# 首页仪表盘

## 核心目标
首屏让用户看到平台状态 + 快速发起常用操作。

## 线框

```
┌───────────────────────────────────────────────────────────────┐
│  仪表盘                                                       │
├───────────────────────────────────────────────────────────────┤
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐         │
│  │ 12       │ │ 2        │ │ 4.2GB    │ │ 全部正常 │         │
│  │ 项目数   │ │ 进行中   │ │ 缓存占用 │ │ 工具链   │         │
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘         │
│                                                               │
│  最近构建                                  [ + 新建项目 ]    │
│  ┌───────────────────────────────────────────────────────┐   │
│  │ MyAwesomePlugin    ● success   2 分钟前   [详情]      │   │
│  │ ProxyChat          ● running   正在构建   [查看日志]  │   │
│  │ EconomyMod         ✗ failed    8 分钟前   [重试]      │   │
│  └───────────────────────────────────────────────────────┘   │
│                                                               │
│  最新公告 / 提示                                              │
│  ⚠ 检测到 BASE_URL 是 https，建议启用认证。 [前往设置]      │
└───────────────────────────────────────────────────────────────┘
```

## 关键组件

- 4 个 KPI 卡（项目数 / 进行中 / 缓存占用 / 工具链状态）
- 最近 5 条构建列表（StatusBadge + 时间 + 操作）
- 系统提示 banner（auth / 工具链缺失 / 产物超额 / etc.）

## 数据来源

- `GET /api/projects?summary=true&limit=5`
- `GET /api/builds?status=running,recent&limit=5`
- `GET /api/cache/status`
- `GET /api/toolchains` 的健康聚合
- 客户端环境变量探测（baseUrl 是否 https）

## 状态

- **加载**：4 个骨架卡 + 列表骨架
- **空**：未有项目时显示大 CTA "开始你的第一个 MC 插件"
- **错误**：每个区域独立 `[ErrorBoundary]`，单点失败不影响其他

## 关键交互

| 触发 | 行为 |
|------|------|
| 点击 KPI "项目数" | 跳转 `/projects` |
| 点击 KPI "进行中" | 跳转 `/projects?filter=running` |
| 点击构建条目 "[详情]" | 跳转 `/projects/:id/builds/:buildId` |
| 点击 "+ 新建项目" | 跳转 `/projects/new` |
| 系统提示 [前往设置] | 跳转 `/settings#auth` |

## 性能

- KPI 卡并发请求（`useQueries`）
- SSE 订阅 running 构建，状态实时变化
