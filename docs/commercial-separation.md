# Commercial Separation

## 当前仓库的开源承诺

本仓库（MC-AI-ForgeLab）完整采用 **AGPL-3.0-only** 开源，不包含任何闭源商业功能。

## 不在本仓库实现的内容

本仓库**不会**实现以下内容：

- 商业授权校验
- License server / License key 验证
- Pro Edition / Cloud Edition 功能门控
- 付费墙 / 破解检测
- 远程授权校验
- 闭源功能开关

## 未来商业功能的处理方式

如果未来需要开发商业功能（如 SaaS 托管、云端构建、企业管理后台、计费系统等），将在**独立的私有仓库或独立服务**中进行，不提交到本 AGPL 开源仓库。

这确保：
1. 本仓库始终是完整可用的开源版本
2. 社区贡献不会被用于闭源商业产品
3. AGPL 许可证的精神得到尊重
